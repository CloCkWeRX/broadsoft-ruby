Class Broadworks
In: broadsoft.rb
Parent: Object

This is the main class for the library. I‘ve only tested this with one instance running, although I cannot see any other reason that it shouldn‘t work.

Example of use: 

 require 'rubygems'
 require "broadsoft.rb"

 # Startup the Broadworks stack.
 bs = Broadworks.new("ews.ihs.broadsoft.com", "2208", "thowe1@broadsoft.com", "Password1")

Public Class methods

new(host, port, call_client, password, logger=nil)

The initialization function. Host refers to the IP address of the broadworks OCS, port to it‘s control port. call_client and password are assigned to the user we are monitoring.
